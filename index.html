    Frequently Asked Questions

    SteamWindows

    Installing via Steam is the most common method. Your MBII folder lives inside your Jedi Academy Steam installation.

    • 1Open Steam → Library → right-click Star Wars Jedi Knight: Jedi Academy → Manage → Browse local files.
    • 2This opens the Jedi Academy folder. Navigate into the GameData subfolder.
    • 3Open the MBII folder inside GameData. This is where all .pk3 files go.
    • 4Copy all .pk3 files you downloaded (or run the setup wizard!) and paste them into that MBII folder.
    • 5Launch Movie Battles II. Done — the maps will be available on any server running them.
    Typical Steam path C:\Program Files (x86)\Steam\steamapps\common\Jedi Academy\GameData\MBII\

    💡 Tip: Use the Setup Wizard above — it auto-finds this path for you.

    GOGOther Platforms

    The process is the same regardless of where you bought the game — you just need to find the MBII folder:

    • 1Open GOG Galaxy → click on Star Wars Jedi Knight: Jedi Academy → click the folder icon or Manage Installation → Show folder.
    • 2Navigate to GameData\MBII inside that folder.
    • 3Paste all downloaded .pk3 files into the MBII folder.
    • 4Launch MBII and you're good to go.
    Common GOG paths C:\GOG Games\Star Wars Jedi Knight - Jedi Academy\GameData\MBII\
    D:\GOG Games\Star Wars Jedi Knight - Jedi Academy\GameData\MBII\

    If you installed to a custom location, just search your PC for MBII as a folder name and drop the pk3 files inside it.

    Movie Battles II (MBII) is a free, community-made total conversion mod for Star Wars Jedi Knight: Jedi Academy. It features class-based multiplayer, lightsaber combat, and iconic Star Wars locations from the films.

    • 1Own a copy of Star Wars Jedi Knight: Jedi Academy (available on Steam, GOG, or other platforms).
    • 2Download the Movie Battles II launcher from moviebattles.org.
    • 3Run the launcher — it will auto-install MBII into your Jedi Academy folder.
    • 4Install Acrowe's custom content using the setup wizard above, then connect to a KOTOR-map server!

    A few common things to check:

    • !Make sure the .pk3 files are in the MBII folder, not the base GameData or base folder.
    • !Make sure you did not extract the .pk3 files — they are archives that the engine reads directly.
    • !Restart Movie Battles II completely after adding files.
    • !If connecting to a server, the map must be running on the server — you can't access these maps in the menus unless a server loads them.

    Still having issues? Join the Discord and ask in the support channel.
